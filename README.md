# Sales Data Analysis

## Tools Used
Python, Pandas, Matplotlib, Tableau

## Dataset
Superstore Sales Dataset — 9,800 records

## What I Did
- Cleaned and processed 9,800+ sales records
- Analyzed sales by Category, Region, and Month
- Built interactive Tableau dashboard with 3 visualizations

## Key Findings
- Technology is the highest revenue category
- West region is most profitable
- Peak sales occur in Q4 (Oct-Dec)
